The objective of this lab is to identify a machine learning problem, select an open-source dataset, and understand the basic structure of the data before model training.

Dataset
- Name: Iris Dataset
- Source: UCI Machine Learning Repository
- Type: Open-source dataset
- Problem Type: Supervised Classification

Problem
- Problem Type: Classification
- Learning Type: Supervised Learning
- Target Variable: target (flower species)
- Goal: Predict the iris flower species based on its measurements.
